預測方法

兩種獨立的路徑分類預測方法，可分別執行或組合比較

Combined RRF

Combined RRF（KNN + DTW + Rule-Based 排名融合）

結合三組獨立的排名信號（KNN 特徵距離、DTW 時序對齊、Rule-Based 幾何規則），使用 Reciprocal Rank Fusion 融合後進行加權投票預測。

三組排名信號

A
KNN 排名（摘要特徵距離）
提取 11 維特徵向量：min_distance, closest_lat/lon, approach_heading, mean_wind, max_wind, mean_pressure, track_length, duration_hours, speed, curvature。
StandardScaler 標準化後計算歐式距離，得出排名 (rank_knn)。
權重: α = 0.13
B
DTW 排名（時序路徑對齊）
對 500km context window 內的 4 維時序 [r, θ, wind, pressure] 做 Dynamic Time Warping。
環形角度距離： d(θ₁,θ₂) = min(|θ₁-θ₂|, 2π-|θ₁-θ₂|)
物理標準化： [r/300, θ/π, wind/100, pressure/50]
權重： [1.0, 1.0, 1.0, 0.5] (r, θ, wind, pressure)
Sakoe-Chiba Band： 30% 限制時間彎曲量
權重: w_dtw = 0.62
C
Rule-Based 排名（規則分類優先度）
使用 CWA 分類規則對所有颱風預先分類。
排名規則： 與查詢颱風同分類的候選排在前面 (rank 0~N-1)，不同分類排在後面 (rank N~M-1)。
這樣使 RRF 傾向選擇路徑型態相同的歷史颱風。
權重: rule_weight = 0.25
RRF
Reciprocal Rank Fusion 融合公式
score(t) = 0.13/(60+rank_knn) + 0.62/(60+rank_dtw) + 0.25/(60+rank_rule)
其中：
• 0.13 = α (KNN 權重)
• 0.62 = 1 - α - rule_weight (DTW 權重)
• 0.25 = rule_weight (Rule-Based 權重)
• 60 = rrf_k (平滑常數，避免排名 0 的發散)
最終
Top-K 投票預測
1. 取 RRF 分數最高的 K=5 筆颱風
2. 以距離倒數加權投票：w_i = 1/(distance_i + ε)
3. 選擇加權投票最多的分類作為預測結果

核心參數

參數值說明

w_dtw0.62DTW 排名權重 (1 - 0.13 - 0.25)

Rule-Based Classification（幾何規則分類）

純規則式分類：基於 CWA 官方路徑分類定義，使用軌跡的幾何特徵（穿越判斷、登陸地點、接近方向、位置）直接判定路徑類型。結果 100% 可解釋，無需訓練資料。

分類決策樹（優先順序由高到低）

Priority 1
分類 6：沿東岸/東部海面北上
判斷條件：
• 不穿越台灣西側 (crossed_to_west = False)
• 最近點經度 > 120.8°
• Context 方向 40° ~ 130°（北行）
• 最近點緯度 < 26.0°
• 東側比例 > 30%（軌跡大部分在東側）
例外： 若接近方向強西行(>145°) + 非常近(<60km) → 跳過，按登陸判斷
Priority 2
分類 8：南部海面向東北
判斷條件：
• 進入點緯度 < 22.0°（南部進入）
• Context 方向 < 75°（東北行）
• 不穿越西側
• 東移 > 2.0°（向東移動）
• 最近點經度 > 121.0° + 最近點緯度 < 23.0°
• 距離 > 80km（海面通過）
Priority 3
分類 2/3/4/7/9：有登陸文字時的精確解析
若有登陸地點文字，解析關鍵字判斷登陸區域：
分類 2（北部登陸）： 基隆、宜蘭、蘇澳、北部等
分類 3（中部登陸）： 花蓮、秀姑巒、台中等
分類 4（南部登陸）： 台東、屏東、恆春等
分類 7（海峽北上）： 南方進入(lat<22.5) + 西岸登陸 + 北行>60° + 北移>2.5°
分類 9（特殊路徑）： 登陸點不規則或無法歸類
Priority 4
分類 7：無登陸但南方北上
判斷條件：
• 進入點緯度 < 22.0°（南部進入）
• 最近點經度 < 121.0°（西側）
• Context 方向 70° ~ 140°（北行或西北行）
• 北移明顯（出口緯度 > 進入緯度 + 3.0°）
Priority 5
分類 1/5：海面通過（距離 > 100km）
分類 1（北海面西行）： 距離 > 100km + 最近點緯度 > 23.5°
分類 5（南海面西行）： 距離 > 100km + 最近點緯度 < 23.5°
Priority 6
分類 9：Fallback
以上皆不符合 → 分類為 9（特殊路徑）
